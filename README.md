This project is a static recreation of a modern documentation-focused landing page, inspired by the visual style and layout of the original Mintlify website.
The goal of this project was to closely replicate the design structure, typography, colors, and section layout using only HTML and CSS, strictly following the given constraints.

No JavaScript, frameworks, animations, or responsive behavior were used. The implementation focuses purely on desktop layout fidelity and clean, readable markup.
